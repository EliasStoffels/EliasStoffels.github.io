About

The arcade game Pengo remade in a custom engine written by me.

The game supports:

  • Single player
  • Multiplayer co-op
  • Multiplayer versus

While the engine supports:

  • Keyboard and up to 4 gamepad inputs at the same time
  • Commands
  • Scene Loading
  • Multi-threaded sound system
  • GameObjects/Components
  • Event Listeners
